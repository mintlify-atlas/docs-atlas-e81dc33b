## Overview

The Rock Paper Scissors game features a clean, modern interface with interactive components that provide immediate visual feedback. The UI is built with React and styled using CSS custom properties for consistent theming.

## Action Buttons

The three main action buttons allow you to select rock, paper, or scissors. These circular buttons feature distinct colors and interactive animations.


<Note>
  Both hover and active states scale the button to 130% of its original size, providing clear visual feedback when you interact with the buttons.
</Note>

## Player Display Cards

The Player and Computer cards show the chosen actions and game results using animated icons and color-coded borders.

### Player Component

```javascript App.js
function Player({name = "Player", action="start", winner}) {
  const result = (winner === undefined || winner === "" ) ? "" : 
                  winner === "Player won!" ? "you" : 
                  winner === "Player lost!" ? "pc" : "draw";
  return (
    <div className={`player ${name}-${result}`}>
      <div className={`score ${name}-${result}`}>{`${name}`}</div>
      <div className="action">
        {action && <ActionIcon name={name} action={action} />}
      </div>
    </div>
  )
}
```

## Action Icons

Each action displays as an animated GIF image that enhances the gaming experience.

### Icon Component

```javascript App.js
function ActionIcon({name, action}) {
  const ICONS = {
    "start": {"icon": start, "word": "start"},
    "r": {"icon": rock, "word": "rock"},
    "p": {"icon": paper, "word": "paper"},
    "s": {"icon": scissors, "word": "scissors"},
  }
  const icon = ICONS[action]["icon"]
  const action_word = ICONS[action]["word"]
  const clase = `${action_word}-${name == "Player" ? "user" : "pc" }-icon`
  return (
    <img src={icon} className={clase} alt="icon" />
  )
}
```

### Icon Mirroring

The computer's icons are horizontally flipped to face the player:

```css App.css
.start-pc-icon,
.rock-pc-icon,
.paper-pc-icon,
.scissors-pc-icon {
  -webkit-transform: scaleX(-1);
  transform: scaleX(-1);
}
```

<Info>
  This mirror effect creates a visual "face-off" between you and the computer, making the game feel more interactive and competitive.
</Info>

## Component Hierarchy

The application follows this component structure:

```
App
├── ActionButton (x3: rock, paper, scissors)
│   └── Icon (from react-icons)
├── Player (You)
│   └── ActionIcon
│       └── Animated GIF image
├── Player (Computer)
│   └── ActionIcon

```
